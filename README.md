# Github Commentator

```
GitHub commentator v0.0.1

USAGE:
    pr-commentator [OPTIONS] --branch <Branch> --comment <Comment> --org <GitHub organization> --repo <Repo name> --token <token>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
        --api-url <Api Url>            The Github api base url [default: https://api.github.com/]
    -b, --branch <Branch>              The branch name to retrieve the PR number
        --comment <Comment>            The content of the comment
        --org <GitHub organization>    The Github organisation or username containing the repo
        --repo <Repo name>             The repository name
        --token <token>                The Github token to use
```
